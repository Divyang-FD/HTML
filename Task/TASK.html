3.HTTP/HTTPs

- Hypertext Transfer Protocol, is the protocol used for transferring data between a web browser and a web server.
Hypertext Transfer Protocol Secure, is a secure version of the HTTP protocol used for transmitting data between a web browser and a web server.

4.CLIENT SERVER

A Client-server is a computing architecture where one device (the cilent)request and uses resources or services from another device over a network.

5.DOMAIN

A domain in the context of computing and the internet, refers to a distinct and identifiable group of computers or devices that are part of a large network.

6.HOST

In the context of computing and networking, a host refers to any computer or device that is connected to a network and is capable of sending or receiving data

7.HOSTING

Hosting, in the context of computing and the internet, refers to the practice of providing and maintaining resources, services, or content on a server or a group of servers that are accessible over a network

8.FTP

FTP stands for (File Transfer protocol). It's a standard network protocol used to transfer files between a client and a server on a computer network.

9.PATH

A path, in computing, refers to the specific location or address of a file or directory within a file system.

10.EXTENSION

In computing, refers to the part of a filename that comes after the last dot (.) and indicates the format or type of a file.

11.EDITOR/VSCODE

Editors come in various forms, serving different purposes, such as text editors, code editors, and graphical editors.

12.URL-URI/URN

A URL consists of several components that together define the path to the resources.
A URI, or Uniform Resource Identifier, is a string of characters used to uniquely identify and reference a resource, either on the internet or in another context.
URNs are used to uniquely identify a resource by name, regardless of its location or access method.
